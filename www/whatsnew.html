What's New

  • The Uploader service is now deprecated in favor of FileTransfer. FileTransfer is built on top of Uploader and adds both HTTPS uploading capibility and a new download() function.
  • With BrowserPlus 2.8.1, Chrome is now supported on OS X. Note that we dropped drop & drop support in Chrome since it is natively supported via HTML5.
  • BrowserPlus Docs and Blog have moved to our open source site: http://browserplus.org/.
